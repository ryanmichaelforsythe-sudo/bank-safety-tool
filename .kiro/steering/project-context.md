# Project Context

This project is a take-home assessment for Nymbus's AI-First Product Owner role.

## Product

A consumer-facing web tool that lets users look up a U.S. bank by name and see a plain-English read on the bank's financial health, drawn from public FDIC data. Target user: financially literate consumers and small-business owners with meaningful deposits at community or regional banks.

## Constraints

- Must use the FDIC BankFind Suite public API as the primary data source
- Must be runnable locally and ideally hosted (Vercel or Netlify)
- UI must look intentional — design and UX matter for evaluation
- Spec quality and AI collaboration are reviewed as heavily as code

## Conventions

- Plain-English explanations for every financial indicator shown to users
- Edge cases explicitly handled: bank not found, mergers, missing data, API rate limits
- Mobile-friendly responsive layout

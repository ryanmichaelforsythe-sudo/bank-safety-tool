# Is My Bank Safe?

A consumer-facing web tool that lets users look up any U.S. bank and see a plain-English read on its financial health, drawn from public FDIC data.

> **Note:** Full README content is written in Task 25. Sections will cover: what was built and why, APIs and how they serve the product, run-locally instructions and live demo link, product decisions and reasoning, and what would be added with more time.

---

## Quick Start (once Node is installed)

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm test` | Run test suite (Vitest) |
| `npm run typecheck` | TypeScript type check (no emit) |
| `npm run lint` | ESLint via Next.js |
